# Glossary

## Phase 0

The smallest playable project slice: a single responsive grid scene with orthogonal player movement, map bounds, blockers, and temporary visual placeholders.

## Scene

One Phaser runtime state containing the Phase 0 map and player.

## Grid position

An integer cell address expressed as a zero-based `column` and `row`.

## Blocker

A grid cell the player cannot enter.

## Berlin placeholder

Temporary, locally owned artwork that suggests a Berlin setting without establishing final visual assets or gameplay.
