# Audio sources and licenses

## Original generated SFX

All files under `sfx/` were generated specifically for this project by
`scripts/generate-sfx.mjs`. They are original project assets and may be used,
modified, and redistributed with the game.

Generation format: unsigned 8-bit PCM WAV, mono, 22.05 kHz. The intentionally
limited format supplies the crunchy retro character while keeping the complete
SFX set small.

Modern `*-modern.wav` variants are also original project assets generated by
`scripts/generate-modern-sfx.mjs`. They use signed 16-bit PCM, mono, 44.1 kHz,
with smooth sine/FM synthesis, additive bells, and filtered-noise transients.
They do not use third-party samples.
